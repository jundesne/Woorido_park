## 1. 온보딩 차이점 (모바일 vs 데스크탑)

| 항목 | 모바일 | 데스크탑 |
|------|--------|---------|
| 스플래시 | 2초 로고 | 없음 (바로 랜딩) |
| 소개 | 스와이프 카드 | 히어로 섹션 + 스크롤 |
| 회원가입 | 풀스크린 | 모달 |
| 진행 표시 | 페이지 전환 | 단계 인디케이터 |
