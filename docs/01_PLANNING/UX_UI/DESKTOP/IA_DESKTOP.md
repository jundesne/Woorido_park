# WOORIDO 데스크탑 IA 명세서 v1.0

> **Platform:** Desktop (Web)
> **Breakpoint:** ≥ 1024px
> **Last Updated:** 2026-01-08

---

## 1. 네비게이션 구조

### 1.1 탑 네비게이션

```
┌────────────────────────────────────────────────────────────────────────┐
│  🏠 WOORIDO   |   홈   탐색   내 모임   |   🔍 검색   ₩500,000   🔔   👤  │
└────────────────────────────────────────────────────────────────────────┘
```

| 요소 | 위치 | 설명 |
|------|------|------|
| 로고 | 좌측 | 홈으로 이동 |
| 메인 메뉴 | 중앙 좌측 | 홈, 탐색, 내 모임 |
| 검색 | 중앙 우측 | 글로벌 검색 |
| 잔액 | 우측 | 어카운트 바로가기 |
| 알림 | 우측 | 드롭다운 알림 |
| 프로필 | 우측 | 드롭다운 메뉴 |

### 1.2 사이드바 (모임 상세)

```
┌─────────────┐
│  📰 피드    │  ← 선택됨
│  📅 모임    │
│  📊 장부    │
│  🗳️ 투표    │
│  👥 멤버    │
│  ⚙️ 설정    │
└─────────────┘
```

## 3. 모달 레이아웃

| 유형 | 크기 | 위치 | 사용처 |
|------|------|------|--------|
| 소형 | 400px | 중앙 | 확인/취소 |
| 중형 | 600px | 중앙 | 가입 플로우, 투표 |
| 대형 | 800px | 중앙 | 게시글 작성 |
| 사이드 패널 | 400px | 우측 | 알림, 상세 정보 |
