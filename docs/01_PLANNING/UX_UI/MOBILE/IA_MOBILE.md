> **Platform:** Mobile (iOS/Android)
> **Breakpoint:** < 768px
> **Last Updated:** 2026-01-08

---

## 1. 네비게이션 구조

### 1.1 바텀 네비게이션

```
┌─────────────────────────────────────────────────────┐
│   🏠 홈    🔍 탐색    ➕ 생성    📋 내 모임    👤 MY  │
└─────────────────────────────────────────────────────┘
```

| 탭 | Route | 아이콘 | 설명 |
|----|-------|--------|------|
| 홈 | `/` | 🏠 | 통합 피드 + 추천 |
| 탐색 | `/discover` | 🔍 | 챌린지 검색 |
| 생성 | `/groups/create` | ➕ | FAB 스타일 |
| 내 모임 | `/my-groups` | 📋 | 가입한 챌린지 |
| MY | `/mypage` | 👤 | 어카운트 + 설정 |

### 1.2 헤더

| 화면 | 좌측 | 중앙 | 우측 |
|------|------|------|------|
| 홈 | 로고 | - | 잔액 + 알림 |
| 모임 상세 | ← 뒤로 | 모임명 | ⋮ 더보기 |
| 설정 | ← 뒤로 | 설정 | - |

## 2. 화면별 레이아웃

### 2.1 홈 (`/`)

```
┌─────────────────────────────────────┐
│  헤더 (48px)                        │
├─────────────────────────────────────┤
│  미결 알림 배너 (옵션)               │
├─────────────────────────────────────┤
│  통합 피드 (스크롤)                  │
│  ├── 피드 카드 1                    │
│  ├── 피드 카드 2                    │
│  └── ...                           │
├─────────────────────────────────────┤
│  나의 모임 (가로 스크롤)             │
├─────────────────────────────────────┤
│  추천 챌린지                        │

│  바텀 네비 (56px)                   │
└─────────────────────────────────────┘
```

┌─────────────────────────────────────┐

## 3. 모달/시트 레이아웃

### 3.1 바텀시트

| 유형 | 높이 | 사용처 |
|------|------|--------|
| 소형 | 30% | 필터, 액션 선택 |
| 중형 | 50% | 투표 참여, 댓글 |
| 대형 | 80% | 가입 플로우, 충전 |

### 3.2 모달

| 유형 | 크기 | 사용처 |
|------|------|--------|
| 확인 | 280px | 삭제 확인, 탈퇴 확인 |
| 입력 | 전체 | 게시글 작성 |

## 4. 간격/크기 규칙

| 요소 | 값 |
|------|-----|
| 화면 패딩 | 16px |

| 버튼 높이 | 48px |
| 입력 필드 높이 | 44px |
| 아이콘 터치 영역 | 44x44px |

## 5. 제스처

| 제스처 | 동작 |
|--------|------|
| Pull-to-refresh | 피드 새로고침 |
| Swipe left | 삭제 (내 댓글) |
| Long press | 복사/신고 메뉴 |
| Pinch | 이미지 확대 |
