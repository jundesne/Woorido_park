# Part 3: 주요 기능 (Framer 발표용)

## 섹션 제목
**CORE FEATURES**

## 부제목
5가지 핵심 기능 도메인

## 리드 문구
우리두는 '신뢰의 기술적 구현'을 위해 5가지 핵심 기능을 중심으로 설계되었습니다.
단순 기능 나열을 넘어, 사용자가 '돈을 맡기고 모임을 즐기는' 전체 여정을 기술로 구현했습니다.

---

## 기능 1: 신뢰 기반 자금 관리
### 아이콘 제안: 🛡️ 또는 🔒

**핵심 가치**
계주가 돈을 가지고 도망갈 수 없다

**주요 기능**
• 선충전 락 시스템: 보증금 1개월치 선입금으로 이탈 방지
• 투표 기반 출금: 멤버 과반수 또는 2/3 승인 필요
• 실시간 장부 공개: 1원 단위까지 투명 공개
• 3단계 누진 수수료: 1% / 3% / 1.5%

**기술 스택**
Spring Boot, Django, Recharts

**경쟁 우위**
토스 모임통장은 계주 독단 출금 가능 → WOORIDO는 멤버 전원 동의 없이 1원도 못 빼냄

---

## 기능 2: SNS 기반 커뮤니티
### 아이콘 제안: 💬 또는 📱

**핵심 가치**
돈 관리 + 소통이 한 곳에서

**주요 기능**
• 피드 작성/댓글/좋아요: 인스타그램 UI/UX
• 이미지 업로드: S3 연동
• Optimistic UI: 좋아요 즉시 반영
• 실시간 알림: 댓글/좋아요 Push

**기술 스택**
React 18, TanStack Query, Framer Motion

**경쟁 우위**
카카오톡은 채팅만 가능, 회비 수금 불가 → WOORIDO는 SNS + 금융 = All-in-One

---

## 기능 3: 스마트 온보딩 및 추천
### 아이콘 제안: 🎯 또는 🔍

**핵심 가치**
나와 맞는 모임을 10초 만에 찾는다

**주요 기능**
• 신규 유저 온보딩: 웰컴 카드 + 첫 충전 유도
• 카테고리 큐레이션: 술/여행/게임/독서/운동 등 12개
• 인기 모임 추천: 가입자 수 + 활성도 기반
• Empty State CTA: 모든 빈 화면에 행동 유도

**기술 스택**
Oracle 21c

**확장 계획**
Year 2: Elasticsearch 도입 (실시간 검색, 자동완성)

---

## 기능 4: 멤버십 및 접근 제어
### 아이콘 제안: 👥 또는 🚪

**핵심 가치**
내 모임에 안 맞는 사람은 들어올 수 없다

**주요 기능**
• 가입 제한 설정: 나이/성별/지역/최소 참여 횟수
• 신청 승인/거절: 계주 또는 운영진 권한
• 부적격 멤버 추방 투표: 멤버 2/3 동의 시 강제 퇴출
• 보증금 몰수 정책: 규칙 위반 시 일부 환수

**법적 안전장치**
개인정보법 준수, 투표 기록 익명 보관

**경쟁 우위**
토스는 누구나 가입 가능 → WOORIDO는 엄격한 진입 장벽으로 신뢰 커뮤니티 형성

---

## 기능 5: 일정 및 참가비 관리
### 아이콘 제안: 📅 또는 ⏰

**핵심 가치**
오프라인 모임 일정을 앱에서 관리

**주요 기능**
• 인터랙티브 달력: 납입일 + 모임 일정 자동 연동
• 참가 여부 집계: 참가/불참/미응답 실시간 표시
• 참가비 자동 계산: 참석자만 분담 (불참자 제외)
• 리마인더 알림: D-3, D-1, D-Day Push

**기술 스택**
Radix UI Calendar, TanStack Query

**경쟁 우위**
기존은 카카오톡 + 구글 캘린더 분리 → WOORIDO는 일정 + 결제 통합

## 발표 스크립트 (3분)

**도입 (30초)**
"Part 3, 주요 기능입니다. 우리두는 단순한 모임 관리 앱이 아닙니다.
'신뢰'를 기술적으로 구현한 플랫폼입니다. 5가지 핵심 기능을 소개하겠습니다."

**기능 1 (30초)**
"첫 번째, 신뢰 기반 자금 관리입니다. 토스 모임통장의 가장 큰 문제는 계주가
독단으로 출금할 수 있다는 점입니다. 우리두는 멤버 전원의 투표 없이는
1원도 출금할 수 없습니다. 이것이 우리의 핵심 차별점입니다."

**기능 2 (30초)**
"두 번째, SNS 기반 커뮤니티입니다. 카카오톡으로 채팅하고, 토스로 돈 관리하는
번거로움을 해결했습니다. 피드, 댓글, 좋아요까지 인스타그램처럼 쓸 수 있고,
회비 수금도 한 곳에서 가능합니다."

**기능 3 (20초)**
"세 번째, 스마트 온보딩입니다. 신규 유저가 10초 만에 자기에게 맞는 모임을
찾을 수 있도록, 12가지 카테고리 큐레이션과 인기 모임 추천 기능을 제공합니다."

**기능 4 (20초)**
"네 번째, 멤버십 접근 제어입니다. 내 모임에 맞지 않는 사람은 아예 들어올 수
없습니다. 나이, 성별, 지역 제한 설정이 가능하고, 부적격 멤버는 투표로 추방할 수 있습니다."

**기능 5 (20초)**
"마지막, 일정 관리입니다. 카카오톡과 구글 캘린더를 오가는 불편함을 없앴습니다.
모임 일정과 참가비 계산이 자동으로 연동됩니다."

**마무리 (30초)**
"5가지 기능 모두 하나의 목표를 향합니다. '신뢰'. 우리두는 기술로 신뢰를 만듭니다.
다음 Part 4에서 이 기능들을 어떻게 9주 만에 구현할지 보여드리겠습니다."
