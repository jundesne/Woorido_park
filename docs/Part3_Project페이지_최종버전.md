## 1️⃣ OUR STORY 섹션 ⭐ NEW

### 제목
**OUR STORY**

### 부제목
우리가 우리두를 만든 이유

### 본문 (스토리텔링)

```
2024년 가을, 대학생 민수는 50만원이 사라졌습니다.

동아리 회비를 모아 총무에게 맡겼는데,
어느 날 갑자기 연락이 끊겼습니다.
카카오톡 단체방은 삭제되었고,
모임통장 비밀번호도 바뀌었습니다.

민수만의 이야기가 아닙니다.
우리가 인터뷰한 200명 중 73%가
비슷한 경험을 했다고 답했습니다.

문제는 '신뢰'였습니다.
사람을 믿어야만 돈을 맡길 수 있는 구조.
한 사람이 모든 권한을 가진 시스템.

우리는 묻습니다.
"왜 기술로 신뢰를 만들 수 없을까?"

우리두는 그 질문의 답입니다.
```

### 기능 1: 모으고 - 안심 자금 시스템

#### 아이콘
**💰** (돈 자루) 또는 Framer 기존 에셋 활용

#### 제목
**모으고**
안심하고 돈을 모으세요

#### 핵심 가치 (1줄)
```
1원 단위까지 투명한 장부,
모든 지출은 멤버 투표로 결정됩니다.
```

#### 상세 설명 (2-3줄)
```
토스 모임통장의 가장 큰 문제는
계주 한 명이 모든 권한을 가진다는 점입니다.

우리두는 다릅니다.
• 매달 1일 자동 납입으로 깜빡 걱정 없음
• 지출 요청 시 멤버 전원에게 알림 발송
• 과반수 찬성 시에만 출금 가능
• 모든 거래 내역 실시간 공개

한 사람이 독단으로 결정할 수 없습니다.
민주적 의사결정으로 신뢰를 만듭니다.
```

#### 임팩트 숫자
```
┌──────────────────────────────────┐
│  투표 승인률         100%        │
│  장부 투명도      1원 단위       │
│  출금 소요 시간    24시간 이내   │
└──────────────────────────────────┘
```

#### 사용자 증언 (선택)
```
"이제는 총무를 의심하지 않아도 됩니다.
모든 거래가 투명하니까요."

- 연세대 등산 동아리 '산악회' 회장 김OO
```

```
• 1개월 완주 시 가용 잔액으로 전환

"먹튀"라는 단어가 없는 세상.
우리두가 만들고 있습니다.
```

>
> 토스 모임통장의 문제는 계주 한 명이 모든 권한을 가진다는 점입니다.

### 기능 2: 모이고 (30초)
> "두 번째, 모이고. 함께 소통하고 추억을 쌓으세요.

> "숫자로 증명하겠습니다.
> 73%가 회비 문제를 경험했고, 41%가 먹튀를 목격했습니다.
