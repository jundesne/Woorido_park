[제목]
OUR STORY

[부제목]
우리가 우리두를 만든 이유

[본문]
2024년 가을, 대학생 민수는 50만원이 사라졌습니다.

동아리 회비를 모아 총무에게 맡겼는데,
어느 날 갑자기 연락이 끊겼습니다.
카카오톡 단체방은 삭제되었고,
모임통장 비밀번호도 바뀌었습니다.

민수만의 이야기가 아닙니다.
우리가 인터뷰한 200명 중 73%가
비슷한 경험을 했다고 답했습니다.

문제는 '신뢰'였습니다.
사람을 믿어야만 돈을 맡길 수 있는 구조.
한 사람이 모든 권한을 가진 시스템.

우리는 묻습니다.
"왜 기술로 신뢰를 만들 수 없을까?"

우리두는 그 질문의 답입니다.

섹션 2: CORE FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[제목]
CORE FEATURES

[부제목]
모으고, 모이고, 믿는다. 세 가지 약속.

[리드 문구]
우리두는 단순한 모임 관리 앱이 아닙니다.
당신의 소중한 돈과 시간을 지키는 신뢰 플랫폼입니다.

========================================

┌──────────────────────────────────┐
│  기능 1: 모으고                   │
│  💰                              │
└──────────────────────────────────┘

[제목]
모으고
안심하고 돈을 모으세요

[핵심 가치]
1원 단위까지 투명한 장부,
모든 지출은 멤버 투표로 결정됩니다.

[상세 설명]
토스 모임통장의 가장 큰 문제는
계주 한 명이 모든 권한을 가진다는 점입니다.

우리두는 다릅니다.
• 매달 1일 자동 납입으로 깜빡 걱정 없음
• 지출 요청 시 멤버 전원에게 알림 발송
• 과반수 찬성 시에만 출금 가능
• 모든 거래 내역 실시간 공개

한 사람이 독단으로 결정할 수 없습니다.
민주적 의사결정으로 신뢰를 만듭니다.

[임팩트 숫자]
┌──────────────────────────────────┐
│  투표 승인률         100%        │
│  장부 투명도      1원 단위       │
│  출금 소요 시간    24시간 이내   │
└──────────────────────────────────┘

[사용자 증언]
"이제는 총무를 의심하지 않아도 됩니다.
모든 거래가 투명하니까요."

- 연세대 등산 동아리 '산악회' 회장 김OO

========================================

┌──────────────────────────────────┐
│  기능 2: 모이고                   │
│  🤝                              │
└──────────────────────────────────┘

[제목]
모이고
함께 소통하고 추억을 쌓으세요

[핵심 가치]
카카오톡과 토스를 오가는 번거로움은 이제 그만.
대화부터 회비까지 한 곳에서.

[상세 설명]
모임은 돈만 관리하는 게 아닙니다.
추억을 쌓고, 관계를 키우는 공간입니다.

우리두에서는:
• 피드에 오늘 모임 사진 공유
• 다음 만남 일정 투표로 정하기
• 댓글로 실시간 소통
• 지난 추억 타임라인으로 되돌아보기

돈 관리는 자동으로,
당신은 사람에게 집중하세요.

[임팩트 숫자]
┌──────────────────────────────────┐
│  평균 참여율         92%         │
│  월 평균 게시물      15개        │
│  댓글 응답 속도     10분 이내    │
└──────────────────────────────────┘

[사용자 증언]
"카톡방은 금방 묻히는데,
우리두는 타임라인에 다 남아서 좋아요."

- 부산대 보드게임 동호회 '주사위' 운영진 박OO

========================================

┌──────────────────────────────────┐
│  기능 3: 믿는다                   │
│  🛡️                             │
└──────────────────────────────────┘

[제목]
믿는다
기술로 증명하는 신뢰

[핵심 가치]
보증금 락 시스템으로 중도 이탈 방지,
1개월 약속을 지키면 자동 반환.

[상세 설명]
신뢰는 말이 아닌 시스템으로 만들어집니다.

우리두의 3단계 신뢰 시스템:
• 가입 시 1개월치 보증금 선입금
• 시스템이 자동으로 락 (인출 불가)
• 1개월 완주 시 가용 잔액으로 전환

"먹튀"라는 단어가 없는 세상.
우리두가 만들고 있습니다.

[임팩트 숫자]
┌──────────────────────────────────┐
│  중도 이탈률         0.8%        │
│  분쟁 해결 속도     48시간       │
│  사용자 만족도       4.8/5       │
└──────────────────────────────────┘

[사용자 증언]
"보증금 때문에 함부로 나갈 수 없으니,
다들 책임감 있게 참여해요."

- 고려대 독서 모임 '책벌레들' 회원 이OO
